# Istanbul 四日旅行网站

这是一个无需后端、可完全本地运行的静态旅行网站。

## 启动

在本目录执行：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

也可以直接双击 `index.html` 浏览。建议使用本地服务器，以获得一致的页面跳转体验。

## 内容结构

- 首页：Day 0–Day 3 总览、按天筛选、展开时间线
- 行程打卡：勾选状态保存在浏览器本地
- 景点详情：费用、开放时间、建议停留、拍照点、注意事项、交通与官网
- 响应式布局：适配桌面、平板和手机
